5. Seguridad

Este sitio web utiliza protocolo HTTPS (cifrado SSL) para garantizar que la comunicación entre tu navegador y el servidor sea segura y los datos no puedan ser interceptados.

6. Confidencialidad

Tus datos no serán cedidos a terceros, salvo obligación legal.
